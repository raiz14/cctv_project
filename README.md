# Код вычислений третьей главы магистерской диссертации Серебренников Д. Е.
В первом документе проводится создание общего датафрейма всех пространственных объектов OpenStreetMap

Во втором - пространственное объединение информации пространственном контексте расположения камер наблюдения и объектов, рядом с которыми нет камер

В третьем файле - проводится LASSO-регуляризация и cloglog-регрессия, а также выводятся таблицы метрик качества модели и различные описательные таблицы

В четвертом документе создаются файлы для последующего создания карты на платформе carto.com
